# Документация Главной Страницы

Главная страница (`/app/page.tsx`) является входной точкой для пользователей и представляет собой лендинг, состоящий из нескольких ключевых секций. Страница полностью адаптивна и поддерживает мультиязычность.

## Структура Страницы

Страница состоит из следующих компонентов, расположенных последовательно:

1. **Hero** (`components/sections/Hero.tsx`)
2. **ProgramsSection** (`components/sections/ProjectsSection.tsx`)
3. **PhilosophySection** (`components/sections/NewsSection.tsx`)
4. **ContactSection** (`components/sections/ContactSection.tsx`)

---

## Компоненты

### 1. Hero Section
**Файл:** `components/sections/Hero.tsx`

Первый экран сайта. Содержит фоновое видео и статистику школы.

**Особенности:**
- **Фон:** Видео (`/videos/Школа.mp4`) с фоллбэком на изображение. Используется оверлей `.hero-overlay` для затемнения.
- **Статистика:** Три счетчика с анимацией набора чисел (`useCountUp`):
  - Год основания (2016)
  - Количество учеников (1000+)
  - Количество кружков (15+)
- **Локализация:** Все текстовые данные берутся из контекста `useLanguage`.
- **Адаптивность:** 
  - На мобильных: "Год основания" сверху, остальные два блока снизу в ряд.
  - На десктопе: Все три блока в один ряд.

### 2. Programs Section (Грани Образования)
**Файл:** `components/sections/ProjectsSection.tsx`
**Экспортируется как:** `ProgramsSection`

Секция, демонстрирующая образовательные программы и возможности школы.

**Особенности:**
- **Контент:** Массив `programs` из 6 элементов (Junior, Middle, Senior, Global Network, Life at Intellect, Admissions).
- **Дизайн:** Карточки с изображениями, бейджами (возраст/тип) и описанием, появляющимся при наведении (hover).
- **Анимации:** 
  - `fadeInDown`, `fadeIn`, `fadeInUp` для заголовков и описания при появлении.
  - Hover-эффекты для карточек (подъем изображения, появление текста).
- **Навигация:** Каждая карточка содержит ссылку (`Link`) на соответствующую страницу программы.

### 3. Philosophy Section (Философия Образования)
**Файл:** `components/sections/NewsSection.tsx`
**Экспортируется как:** `PhilosophySection`

Секция, описывающая ключевые методики обучения.

**Особенности:**
- **Контент:** 3 ключевых аспекта (Память, Критическое мышление, Геймификация).
- **Локализация:** Данные загружаются через `t.philosophy`.
- **Интерактивность:** Hover-эффекты с увеличением изображений.
- **Дизайн:** Сетка карточек с закругленными углами и тенями.

### 4. Contact Section (Контакты)
**Файл:** `components/sections/ContactSection.tsx`

Секция с формой записи и контактной информацией.

**Особенности:**
- **Форма:** 
  - Поля: Имя, Телефон, Email.
  - Состояние управляется через `useState`.
  - Обработчик `handleSubmit` (на данный момент выводит данные в консоль).
- **Информация:** Телефоны, адреса (два филиала), ссылка на Instagram.
- **Стилизация:** Поля ввода с фокусом фирменного цвета (`navy-900`).

---

## Используемые Хуки и Контексты

- **`useLanguage`** (`contexts/LanguageContext`): Предоставляет переводы текстов (`t`). Используется в Hero и Philosophy секциях.
- **`useCountUp`** (`hooks/useCountUp`): Кастомный хук для анимации числовых значений в Hero секции.

## Стилизация

- Используется **Tailwind CSS**.
- Основные цвета: 
  - `navy-900` (#0B1B3D) как основной фирменный цвет.
  - Градиенты и glassmorphism эффекты (`backdrop-blur`).
- Шрифты:
  - `IBM Plex Serif` для крупных цифр и акцентных заголовков.
  - `Cormorant` для подписей в статистике.
  - `Inter` / `Montserrat` для основного текста.
