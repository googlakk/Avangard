# Инструкция для Агента: Переделка Страницы

Скопируйте эту инструкцию и отправьте агенту вместе с указанием страницы.

---

## Промпт для Агента:

```
Переделай страницу [УКАЖИТЕ ПУТЬ, например: app/programs/senior/page.tsx] следуя этим правилам:

1. ОБЯЗАТЕЛЬНО прочитай эти документы ПЕРЕД началом работы:
   - docs/DESIGN_SYSTEM.md
   - docs/UX_GUIDELINES.md
   - docs/main_page_documentation.md (для reference главной страницы)

2. Проблема: Текущая страница выглядит как PDF-брошюра (много одинаковых секций подряд, нет интерактивности).

3. Требования к новой версии:
   ✅ Максимум 4 крупных секции (Hero → Value → Details → CTA)
   ✅ Используй ТАБЫ или АККОРДЕОНЫ для группировки однотипного контента
   ✅ Минимум 2 CTA (Call-to-Action) кнопки: одна в Hero, одна в конце
   ✅ Hover-эффекты на всех интерактивных элементах
   ✅ Точное соответствие DESIGN_SYSTEM.md (шрифты, цвета, компоненты)
   ✅ Белое пространство между секциями (py-16 минимум)
   ✅ ФОТОГРАФИИ КАК ДОКАЗАТЕЛЬСТВО: Каждое утверждение должно подкрепляться фото
      - Hero ОБЯЗАТЕЛЬНО с фоновым изображением/видео
      - Минимум 1 фото на каждые 2 текстовых блока
      - Используй реальные фото школы (или качественные placeholders из Unsplash)
      - Избегай "текстовых стен" без визуала

4. Структура, которую нужно создать:
   - Hero (заголовок + краткое описание + главная CTA)
   - Key Benefits (3 карточки максимум)
   - Detailed Info (используй Tabs для разных категорий)
   - Final CTA Section

5. После создания:
   - Убедись, что страница НЕ требует скролла > 3 экранов
   - Проверь, что все интерактивные элементы работают
   - Сравни дизайн с главной страницей (app/page.tsx) — должен быть визуально единым стилем
```
